Customer Engagement Portal



This is a full-stack web application for managing customer records.

It is built using React for the frontend and Node.js, Express, and PostgreSQL for the backend.



Features

\- View customer list

\- Add new customers

\- Prevent duplicate email entries

\- Real-time UI updates

\- Backend validation and error handling



Technology Used



Frontend

\- React

\- JavaScript

\- HTML/CSS



Backend

\- Node.js

\- Express.js

\- PostgreSQL



How the Project Works

\- Frontend runs on port 3001

\- Backend API runs on port 3000

\- PostgreSQL stores customer data



API Endpoints

\- GET /customers → Fetch all customers

\- POST /customers → Add a new customer



How to Run the Project



Backend

1\. Open terminal

2\. Go to server folder

3\. Run: npm install

4\. Run: node index.js



Frontend

1\. Open another terminal

2\. Go to client folder

3\. Run: npm install

4\. Run: npm start
